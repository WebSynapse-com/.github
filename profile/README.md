
## **websynapse**

🙋‍♀️ 我们是做什么的?

一款加密领域的智能助手，整合断裂的金融数据，结合实时的新闻数据为用户提供定制化的问答。

+ 数据来源: twitter、各大主流新闻平台、链上金融数据

🍿 为什么要这么做?

1. 加密领域的数据断裂非常严重，不同链的交易数据、不同平台的新闻数据，无法真正的仅通过AI的方式聚合起来。对于搜索引擎，在消息时延以及准确性来讲还有非常大的可提升空间。
2. 用户更渴望站在个人的资产情况的基础上与AI进行交流。
3. 目前AI更擅长于回答用户问题，而不是基于当前突出的事件下主动向用户提出见解。
